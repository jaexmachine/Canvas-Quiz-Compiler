# Canvas Quiz Compiler

> A Chrome extension to streamline the creation and compilation of Canvas Quizzes from a simple, structured text format.

## Features

* Compiles structured text into ready-to-use Canvas Quiz questions.
* Supports Multiple-Choice questions (Initial Release).
* Built-in validation for quick error checking.

## Usage

1.  Navigate to the Quiz creation page on Canvas.
2.  Open the extension popup.
3.  Paste your structured quiz content into the text area.
    
    *Example Format:*
    ```
    # Multiple Choice
    What is the capital of France?
    * Paris (Correct)
    * London
    * Rome
    ```
    
4.  Click the **"Compile & Apply"** button.
