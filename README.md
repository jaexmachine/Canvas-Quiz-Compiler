# Canvas Quiz Compiler

> A Chrome extension to streamline the creation and compilation of Canvas Quizzes from a simple, structured text format.

## Features

* Compiles structured text into ready-to-use Canvas Quiz questions.
* Supports Multiple-Choice questions (Initial Release).
* Built-in validation for quick error checking.

## Getting Started

### Installation

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/YourUsername/canvas-quiz-compiler.git](https://github.com/YourUsername/canvas-quiz-compiler.git)
    ```
2.  **Open Chrome Extensions:**
    * Navigate to `chrome://extensions/` in your browser.
3.  **Enable Developer Mode:**
    * Toggle the "Developer mode" switch on (usually in the top-right).
4.  **Load Unpacked:**
    * Click the "**Load unpacked**" button and select the root folder of the cloned repository.

## Usage

1.  Navigate to the Quiz creation page on Canvas.
2.  Open the extension popup.
3.  Paste your structured quiz content into the text area.
    
    *Example Format:*
    ```
    # Multiple Choice
    What is the capital of France?
    * Paris (Correct)
    * London
    * Rome
    ```
    
4.  Click the **"Compile & Apply"** button.
