# Canvas Quiz Compiler

> A Chrome extension designed to simplify the creation of Canvas LMS quizzes by compiling structured, plaintext question lists directly into the quiz editor interface.

## Features

The Canvas Quiz Compiler is designed to dramatically speed up quiz creation by allowing you to define questions in a simple, readable text format.

* **Fast Compilation:** Converts structured text into Canvas-compatible question data quickly and efficiently.
* **Initial Question Support (TBD):**
    * **Multiple Choice:** Supports defining correct and incorrect answer choices easily.
    * **True/False:** Simple structure for quick True/False statements.
* **Direct Injection:** Automatically inserts the compiled questions directly into the Canvas Quiz creation interface.
* **Simple Interface:** Uses a clean, minimalist text area within the extension popup for user input. 

## Getting Started

To use the Canvas Quiz Compiler, follow these steps to load the extension in Chrome:

### Usage

1.  Navigate to the Quiz creation page within your Canvas course.
2.  Click on the newly installed **Canvas Quiz Compiler** icon in your browser toolbar to open the popup.
3.  Paste your structured quiz content into the text area.
    *Example Format (Multiple Choice):*
    ```
    # MC
    What is the core feature of the Canvas Quiz Compiler?
    * Compiling text (Correct)
    * Grading student work
    * Sending emails
    ```
4.  Click the **"Compile & Apply"** button to generate and insert the questions.
